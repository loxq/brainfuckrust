# Brainfuck interpreter written in plain Rust

## Usage

cargo test

cargo build

cargo run -- code.bf

## Example

```
$ cargo run -- helloworld.bf
    ++++++++++
    [
        >+++++++
        >++++++++++
        >+++
        >+<<<<-
    ]   >++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.
    --------.>+.>.


Hello World!

```
